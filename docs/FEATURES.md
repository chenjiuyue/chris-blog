# Chris Know 功能说明文档

本文档详细介绍博客系统的所有功能特性和使用方法。

---

## 📖 阅读体验功能

### 1. 文章目录导航

**位置**: 文章详情页右侧（大屏幕）

**功能说明**:
- 自动从文章内容提取标题（H1-H6）生成目录树
- 点击目录项平滑滚动到对应章节
- 滚动时自动高亮当前阅读的章节
- 支持多级标题嵌套显示

**技术实现**:
- 使用 `IntersectionObserver` API 监听标题元素可见性
- 为每个标题自动生成唯一 ID
- CSS `position: sticky` 实现固定定位

**适用场景**:
- 长文章阅读
- 技术文档
- 教程类文章

---

### 2. 阅读进度条

**位置**: 页面顶部固定

**功能说明**:
- 实时显示文章阅读进度
- 彩色进度条（蓝色）
- 平滑过渡动画

**技术实现**:
- 监听 `scroll` 事件计算滚动百分比
- 动态设置进度条宽度
- CSS `transition` 实现平滑效果

**计算公式**:
```javascript
progress = (已滚动距离 / 总内容高度) * 100%
```

---

### 3. 返回顶部按钮

**位置**: 页面右下角浮动

**功能说明**:
- 滚动超过 300px 后显示
- 点击平滑滚动到顶部
- Hover 放大动画

**技术实现**:
- 监听 `scroll` 事件控制显示/隐藏
- `window.scrollTo({ top: 0, behavior: 'smooth' })` 平滑滚动
- CSS `transform: scale()` 实现缩放动画

**样式特点**:
- 圆形按钮
- 蓝色背景 + 白色图标
- 阴影效果

## 💬 互动功能

### 4. 文章点赞功能

**位置**: 文章详情页正文下方

**功能说明**:
- 点赞/取消点赞切换
- 点赞数实时更新
- 点赞状态持久化
- 心形图标动画效果

**技术实现**:
- **云函数**: `blog-toggleLike` 处理点赞逻辑
- **数据库**: `blog_likes` 集合记录点赞
- **本地存储**: `localStorage` 保存点赞状态
- **用户标识**: 自动生成匿名用户 ID

**云函数逻辑**:
```javascript
// 检查是否已点赞
if (已点赞) {
  // 删除点赞记录
  // 文章点赞数 -1
  return { liked: false }
} else {
  // 添加点赞记录
  // 文章点赞数 +1
  return { liked: true }
}
```

**数据一致性**:
- 使用云函数原子操作 `_.inc(1)` 和 `_.inc(-1)`
- 避免并发冲突
- 确保数据准确性

---

### 5. 文章收藏功能

**位置**: 文章详情页正文下方（点赞按钮旁）

**功能说明**:
- 收藏/取消收藏切换
- 收藏列表页面查看
- 收藏状态持久化
- 书签图标动画效果

**技术实现**:
- **存储方式**: `localStorage`（纯前端）
- **数据格式**: `{ postId, title, createdAt }[]`
- **收藏页**: `/favorites` 路由

**本地存储优势**:
- 无需后端支持
- 即时响应
- 离线可用
- 隐私保护

**收藏列表功能**:
- 显示收藏文章标题
- 点击跳转到文章详情
- 一键删除收藏
- 显示收藏数量

---

### 6. 评论系统

**位置**: 文章详情页底部

**功能说明**:
- 匿名评论（无需注册）
- 嵌套回复（最多 2 层）
- 实时评论计数

**技术实现**:
- **认证**: CloudBase 匿名登录
- **数据库**: `blog_comments` 集合
- **安全规则**: `read: true, create: auth != null`

**评论字段**:
```typescript
{
  _id: string
  postId: string
  nickname: string
  content: string
  parentId: string  // 回复时填写
  createdAt: string
}
```

---

### 7. 留言板

**位置**: 导航菜单 - 留言板（`/guestbook`）

**功能说明**:
- 访客留言提交
- 留言审核机制
- 留言墙展示
- 昵称记忆功能

**技术实现**:
- **数据库**: `blog_guestbook` 集合
- **审核状态**: `pending`（待审核）、`approved`（已通过）、`rejected`（已拒绝）
- **安全规则**: 仅显示已审核通过的留言

**使用流程**:
1. 填写昵称和留言内容
2. 提交留言（状态为 `pending`）
3. 管理员审核通过
4. 留言显示在留言墙

**昵称记忆**:
- 首次留言后保存昵称到 `localStorage`
- 下次自动填充昵称
- 方便快速留言

**位置**: 首页侧边栏

**位置**: 首页侧边栏、文章详情页

**功能说明**:
- 按分类筛选文章
- 按标签筛选文章
- 标签云展示

**技术实现**:
- **数据库**: `blog_categories` 和 `blog_tags` 集合
- **筛选**: 路由参数 `/category/:slug` 和 `/tag/:slug`

### 11. 全文搜索

**位置**: 首页侧边栏搜索框

**功能说明**:
- 按文章标题搜索
- 实时搜索结果
- 搜索历史记录

**技术实现**:
- 使用正则表达式 `db.RegExp({ regexp: keyword, options: 'i' })`
- 不区分大小写
- 搜索 `blog_posts.title` 字段

**搜索流程**:
1. 输入关键词
2. 按回车或点击搜索按钮
3. 显示匹配的文章列表
4. 点击"返回全部文章"恢复

## 🚀 性能优化功能

### 15. PWA 支持

**功能说明**:
- 离线访问支持
- 添加到主屏幕
- Service Worker 缓存
- 资源预缓存

**技术实现**:
- **插件**: `vite-plugin-pwa`
- **配置**: `vite.config.ts`
- **Manifest**: 应用名称、图标、主题色

**缓存策略**:
- **静态资源**: CacheFirst（优先使用缓存）
- **CloudBase 资源**: NetworkFirst（优先使用网络）
- **字体文件**: 长期缓存（1 年）

**离线体验**:
- 已访问页面离线可用
- 图片、样式自动缓存
- 网络恢复后自动同步
